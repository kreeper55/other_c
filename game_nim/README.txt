Игра Ним
Правила:
поочередно ходят два игрока (игрок - пк), каждый берет из кучки спички (но не больше чем есть в куче)
Выигрывает тот, кто последним заберет спичку

Управление
Цифры на клавиатуре для выбора игры и набора цифр
выход из игры на клавишу q

Ввод производится через пробел 
(например 2 4, где первая цифра это номер кучи, а вторая количество спичек, которые забираем из этой кучи)
