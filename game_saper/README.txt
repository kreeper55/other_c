Управление в меню осуществляется стрелочками
вверх-вниз переключение между элементами меню
стрелка вправо - выбор элемента меню
клавиша ESC - выход из меню
Выбор сложности в меню осуществляется стрелочками
вверх-вниз переключение между элементами и стрелка вправо выбор
при настраиваемом режиме, можно ввести начальные данные через пробел
или же каждый по отдельности (>> number ENTER)
Управление в игре
Чтобы отметить координату, нужно ввести два
двухзначных числа через запятую: число1,число2
чтобы выйти с игрового поля, напишите 'exit'
нажмите клавишу ESC, чтобы вернуться в меню
